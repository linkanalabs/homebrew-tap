# linkanalabs/homebrew-tap

Homebrew tap da [Linkana](https://linkana.com).

## Disponível

### [lk](https://github.com/linkanalabs/cli) — Linkana CLI

```bash
brew install linkanalabs/tap/lk
```

Para atualizar:

```bash
brew upgrade lk
```

Os casks em `Casks/` são gerados e publicados automaticamente pelo
[GoReleaser](https://goreleaser.com) a cada release do repositório de origem —
**não edite na mão**.
